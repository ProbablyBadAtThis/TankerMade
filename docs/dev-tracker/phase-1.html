Foundation Phase

Duration: Weeks 1-3
Total Tasks: 65
Estimated Time: 60-80 hours
Complexity: High (Setup)
Goal: Get basic infrastructure running in both offline and server modes

What you'll build: A working Blazor WASM app that can create, edit, and list projects both offline (IndexedDB) and with a server (JWT auth + API). This is the foundation everything else builds on.
